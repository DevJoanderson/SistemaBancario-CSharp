# Sistema Bancário em C#

Projeto desenvolvido para praticar os fundamentos da linguagem C# e Programação Orientada a Objetos.

## Funcionalidades

* Visualizar saldo
* Realizar depósitos
* Realizar saques
* Consultar histórico de transações
* Visualizar informações da conta
* Persistência de dados em arquivo (`saldo.txt`)

## Tecnologias Utilizadas

* C#
* .NET
* Visual Studio

## Conceitos Praticados

* Classes e Objetos
* Métodos
* Construtores
* Encapsulamento básico
* Estruturas condicionais (`if`)
* Estruturas de repetição (`while`)
* Estrutura de decisão (`switch`)
* Listas (`List<T>`)
* Manipulação de arquivos (`System.IO`)
* Programação Orientada a Objetos (POO)

## Estrutura do Projeto

### Classe Conta

Responsável por representar uma conta bancária.

Atributos:

* Id
* Nome
* Saldo

Métodos:

* Depositar()
* Sacar()

### Classe BancoApp

Responsável pela interação com o usuário através do menu.

Funcionalidades:

* Exibir menu
* Controlar operações bancárias
* Gerenciar histórico
* Salvar e carregar saldo em arquivo

## Como Executar

1. Clone o repositório
2. Abra o projeto no Visual Studio
3. Execute o projeto
4. Utilize o menu para realizar operações bancárias
